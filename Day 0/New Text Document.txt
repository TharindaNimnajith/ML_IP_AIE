Iris flower data set : Classification Problem
				       Features : Sepal length, Sepal width, Petal length, Petal width (4 Features)
					   Data     : All the values of features in the sample (4 * 150 = 600 data) { 2D Array -> data[150][4] }
					   Labels   : Setosa, Virginica, Versicolor (3 Labels)
					   Targets  : No of samples / All the values in the labels (150 targets) { 1D Array -> targets[150] }


ML Categories:

     Supervised ML 	 	 ->  Label is given
			     	         Accuracy is higher

     Unsupervised ML  	 ->  Label is not given
			     		  	 Categorizing / Clustoring into Classes
			     		  	 Letting the machine to predict the number of labels
			    		  	 Self-Learning
			     		   	 More Intelligent

     Semi-Supervised ML  ->  Using both labeled & unlabeled data sets
